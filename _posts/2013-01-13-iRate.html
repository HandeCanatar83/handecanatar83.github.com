iRate sayesinde kullanıcılarımıza uygulamamızı oylatabiliriz.

IOS-Application-Single View Application'ı seçerek yeni bir uygulama oluşturuyoruz veya var olan bir uygulamanızı kullanabilirsiniz.

iRate.h iRate.m Uygulamamıza iRate.h ve iRate.m Classlarını ekliyoruz. Classların güncel halini iRate'in Github projesinden de alabilirsiniz. nicklockwood / iRate

Daha sonra AppDelegate.h classımızda aşağıdaki değişiklikleri yapıyoruz. nedenlerini commentlerde anlattım. :)

{% highlight objc %} //AppDelegate.h #import @class ViewController; //--1 // iRate mesajından sonra hangi view ile uygulamanız başlayacak @interface AppDelegate : UIResponder @property (strong, nonatomic) UIWindow *window; @property (nonatomic, strong) IBOutlet ViewController *viewController;//--2 //ViewController'ı IRate e göstermek için bir property hazırladık. @end {% endhighlight %}

AppDelegate.m classımıza gelip kodlarımızı ekliyoruz.

{% highlight objc %} //AppDelegate.m #import "AppDelegate.h" #import "iRate.h"//--3 #import "ViewController.h"//--4 @implementation AppDelegate @synthesize viewController;//--5 @synthesize window;//--6 + (void)initialize { //--7 [iRate sharedInstance].applicationBundleID = @"com.bundleım.uygulamaismi"; // bundleım : provisioning portalda daha önce oluşturduğunuz Bundle Identifier (App ID Suffix) // uygulamaismi : uygulama ismi işte :) [iRate sharedInstance].onlyPromptIfLatestVersion = NO; //enable preview mode [iRate sharedInstance].previewMode = YES; //Test uygulamasında çalıştığımız için 5 gün beklemeyeceğiz ve iRate hemen çalışacak. [iRate sharedInstance].daysUntilPrompt = 5; [iRate sharedInstance].usesUntilPrompt = 15; //--7 } - (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions { // Override point for customization after application launch. //--8 [window addSubview:viewController.view]; [window makeKeyAndVisible]; //--8 return YES;; } {% endhighlight %}
